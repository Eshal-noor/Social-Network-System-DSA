# 🧠 DSA Social Network Simulator (C++ + Qt)

A complete Data Structures and Algorithms project implemented in C++ with both Console and Qt GUI versions. 

This project simulates a simple social network system using core DSA concepts like graphs, stacks, queues, linked lists, and arrays.

---

## 🛠️ Data Structures Used
* 🟢 **Graph:** Social connections / relationships
* 🟢 **Binary Search Tree:** Data management
* 🟢 **Stack:** Chat/message handling
* 🟢 **Queue:** News feed simulation
* 🟢 **Doubly Linked List:** Navigation system
* 🟢 **Arrays:** User profiles

---

## 💻 Versions

### 1. Console Version
* Simple C++ implementation
* Focuses on logic and DSA concepts
* No GUI

### 2. Qt GUI Version
* Built using Qt framework
* Interactive graphical interface
* Includes:
  * Chat system (stack)
  * News feed (queue)
  * Profile management (array)
  * Graph visualization logic (backend)
